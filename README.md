# 🌐 Kanniyakumarione – CEO Page

CEO-led digital experiences, strategic launch orchestration, and public leadership for [kanniyakumarione.com](https://kanniyakumarione.com/). This site consolidates the CEO narrative with high-impact storytelling, immersive motion, and SEO-ready messaging.

Built with **React**, **Vite**, **Tailwind CSS**, **Framer Motion**, and **Firebase**, with a focus on performant, founder-first interactions.

## 🧰 Tech Stack

### Frontend
- **React** (component-driven UI)
- **Vite** (lightweight bundler + dev server)
- **Tailwind CSS** (utility-first design)
- **Framer Motion** (micro-interactions + section reveals)
- **Lucide Icons**

### Backend / Services
- **Firebase Firestore** (feedback + contact pipeline)
- **Firebase Hosting (optional)**

### Infrastructure & Delivery
- **Vercel** for deploy previews + production push.
- **Firebase Functions** (optional future automation)
- **GitHub Actions** (optional push-based checks)

### SEO & Discoverability
- Meta description, canonical link, and OG tags in `index.html`.
- Structured hero, project, and contact sections to drive engagement.
- Content copy that reinforces the Kanniyakumarione CEO story.

## 🔐 Environment Variables

Create a `.env` file in the root with the keys below (values come from your Firebase console):

```bash
VITE_FIREBASE_API_KEY="..."
VITE_FIREBASE_AUTH_DOMAIN="..."
VITE_FIREBASE_PROJECT_ID="..."
VITE_FIREBASE_STORAGE_BUCKET="..."
VITE_FIREBASE_MESSAGING_SENDER_ID="..."
VITE_FIREBASE_APP_ID="..."
```

### Quick start

1. `git clone <repo>`
2. `npm install`
3. `npm run dev`
4. `npm run build`
5. `npm run preview`
